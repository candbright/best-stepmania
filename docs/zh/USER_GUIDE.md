# Best-StepMania 用户指南（精简版）

本指南与 **v1.0.0** 对齐。版本更新见 [changelog/bsm-v1.0.0.md](../changelog/bsm-v1.0.0.md)。

## 1. 你会用到什么

- 一台可运行桌面应用的电脑（Windows 优先）。
- Node.js 与 Rust（用于本地开发和构建）。
- 歌曲资源：音频 + `*.sm` 或 `*.ssc` 谱面。

## 2. 快速开始

1. 安装依赖：在项目根目录运行 `npm install`。
2. 启动网页调试：`npm run dev`。
3. 启动桌面应用：`npm run tauri:dev`。
4. 构建安装包：`npm run tauri:build`。

注意：不要直接运行 `target/debug/best-stepmania.exe`，它依赖 `tauri:dev` 的本地服务。

## 3. 玩家主流程

### 选歌

- 进入选歌页后可浏览、搜索、筛选、排序。
- 可创建歌曲包并把歌曲整理进包里。

### 游玩

- 进入玩家选项页设置速度、显示和按键。
- 开始后按节奏击打音符，系统按判定窗口给出结果。
- 结束后进入结算页查看分数、评级和命中统计。

### 编辑器

- 从编辑器入口选择歌曲与谱面。
- 在画布中添加、删除、移动音符。
- 可调整节拍与时间轴参数并保存回谱面文件。

## 4. 常用设置

- 音频：主音量、音效音量、预览音量。
- 画面：主题、窗口显示模式、渲染质量。
- 输入：按键映射、快捷键、玩家布局。
- 游戏：判定显示、生命条行为、常用游玩选项。

## 5. 常见问题

### 歌曲列表为空

- 检查导入路径是否正确。
- 确认歌曲目录里有音频和 `*.sm`/`*.ssc`。
- 重新导入一次并观察导入提示。

### 音画不同步

- 先在设置里调音频偏移。
- 关闭高负载程序后再测试。
- 优先用稳定帧率和较低渲染质量做基线测试。

### 键盘没反应

- 打开按键设置确认绑定是否存在冲突。
- 切出输入法或改为英文输入再试。
- 在其它界面测试按键，排除硬件问题。

### 谱面加载失败

- 检查谱面编码和标签格式。
- 确认引用的音频文件存在且路径有效。
- 通过诊断页面导出信息再排查。

## 1. 简介

### 文档与版本

本指南与 **v1.0.0** 对齐。按版本划分的更新摘要见 [changelog/bsm-v1.0.0.md](../changelog/bsm-v1.0.0.md)。

### 什么是 Best-StepMania

Best-StepMania 是一款跨平台节奏游戏，灵感来源于 Pump It Up 和 StepMania。它采用现代化的技术栈，基于 Tauri 的桌面应用程序，并支持标准 StepMania 谱面格式（.sm 和 .ssc 文件）。

主要功能包括：

- 支持 Pump 单人（5 轨道）、Pump 双人（10 轨道）、Pump 合奏和合作模式
- 内置谱面编辑器，支持实时预览
- 多种音符皮肤和视觉主题
- 完善的评分系统，包含舞点（DP）和字母等级
- 歌曲包管理功能，帮助整理您的音乐库
- 自动播放模式，用于练习和谱面预览

### 系统要求

**最低要求：**

- Windows 10/11、macOS 10.15+ 或 Linux（Ubuntu 20.04+）
- 4 GB 内存
- 500 MB 可用磁盘空间（外加歌曲所需空间）
- 键盘或兼容的游戏手柄

**推荐配置：**

- 8 GB 内存
- 独立显卡，以获得更流畅的渲染效果
- 固态硬盘，加快歌曲加载速度
- 低延迟音频输出设备

## 2. 入门

### 首次启动

首次启动 Best-StepMania 时，游戏将：

1. 扫描默认的 `Songs/` 目录中的谱面文件
2. 在用户数据目录中创建配置文件
3. 显示标题画面，显示已加载的歌曲数量

如果未找到歌曲，您将看到相应提示。您可以通过导入歌曲包来添加歌曲（见下文）。

### 导入歌曲和歌曲包

Best-StepMania 支持标准 StepMania 谱面格式。添加歌曲的方法：

1. 在标题画面中，选择 **OPTIONS** > **Manage Song Packs**
2. 点击 **Import Pack** 并选择包含您歌曲的文件夹
3. 每首歌曲应放在独立的子文件夹中，包含：
   - 音频文件（.ogg、.mp3 或 .wav）
   - 谱面文件（.sm 或 .ssc）
   - 可选：横幅图片、背景图片

游戏按"包"（文件夹）组织歌曲。您也可以直接将歌曲放在游戏目录的 `Songs/` 文件夹中。

**支持的谱面格式：**

- `.sm` 文件（StepMania 格式）
- `.ssc` 文件（StepMania SSC 格式，推荐）

### 基本导航

使用以下控件浏览菜单：

| 操作 | 默认按键 |
|------|----------|
| 浏览菜单 | 方向键或鼠标 |
| 确认 / 选择 | `Enter` 或点击 |
| 返回 | `Escape` |

UI 完全支持键盘和鼠标操作。

## 3. 选歌

### 浏览歌曲

在标题画面中，选择 **GAME START** 进入选歌界面。在这里您可以：

- 浏览所有可用歌曲
- 点击歌曲进行预览
- 查看每首歌曲的谱面（难度）
- 查看您的高分和等级

每首歌曲显示：

- 标题和艺术家
- BPM（每分钟节拍数）
- 可用谱面及难度等级
- 包名称

### 搜索和筛选

使用搜索框按以下条件查找歌曲：

- 歌曲标题
- 艺术家名称
- 包名称

点击 **FILTER** 按钮应用筛选：

- **难度范围**：设置最低和最高难度等级
- **Song Pack**：仅显示特定包中的歌曲
- **Steps type**：按游戏模式筛选（单人、双人、合奏、合作）

点击 **Clear** 移除所有筛选条件。

### 排序选项

点击排序下拉菜单，按以下条件排列歌曲：

- **Title**：按字母顺序
- **Artist**：按艺术家名称
- **BPM**：按速度（从慢到快）
- **Pack**：按歌曲包名称

### 创建歌曲包

歌曲包有助于整理您的音乐库。创建新包的方法：

1. 从 **OPTIONS** > **Manage Song Packs**
2. 将您的歌曲放入新文件夹
3. 点击 **Import Pack** 并选择该文件夹

您也可以直接创建新歌曲：

1. 在选歌界面，点击 **Create Song**
2. 填写歌曲元数据（标题、艺术家等）
3. 选择音频文件或使用默认生成的音频
4. 选择要将歌曲放入哪个包

## 4. 游戏

### 控制（10 键 Pump 双人布局）

Best-StepMania 使用 10 轨道布局进行 Pump 双人模式。默认键盘映射为：

| 轨道 | 面板 | 默认按键 |
|------|------|----------|
| 1 | P1 左上 | `Q` |
| 2 | P1 右上 | `E` |
| 3 | P1 中心 | `Space` |
| 4 | P1 左下 | `I` |
| 5 | P1 右下 | `O` |
| 6 | P2 左上 | `Numpad 7` |
| 7 | P2 右上 | `Numpad 8` |
| 8 | P2 中心 | `Numpad 5` |
| 9 | P2 左下 | `Numpad 9` |
| 10 | P2 右下 | `Numpad 0` |

您可以在 **OPTIONS** > **Keyboard** 中自定义这些按键。

### 判定系统（判定时间窗口）

Best-StepMania 使用精确的判定时间窗口来评估您的准确度：

| 判定 | 时间窗口 | DDR 名称 | ITG 名称 |
|------|----------|----------|----------|
| W1 (Marvelous) | ±22.5ms | Marvelous | Fantastic |
| W2 (Perfect) | ±45ms | Perfect | Excellent |
| W3 (Great) | ±90ms | Great | Great |
| W4 (Good) | ±135ms | Good | Decent |
| W5 (Boo) | ±180ms | Boo | Way Off |
| Miss | >180ms | Miss | Miss |

您可以在设置中更改判定命名风格（DDR 或 ITG 风格）。

### 评分（DP 和等级计算）

评分系统使用**舞点（DP）**：

| 判定 | DP 值 |
|------|-------|
| W1 / W2 | +2 |
| W3 | +1 |
| W4 | 0 |
| W5 | -4 |
| Miss | -8 |
| Hold 完成 | +6 |
| Mine 触发 | -8 |

**字母等级：**

| 等级 | DP 百分比 |
|------|-----------|
| SSS | 100% |
| SS | 93% - 99.9% |
| S | 80% - 92.9% |
| A | 65% - 79.9% |
| B | 45% - 64.9% |
| C | 20% - 44.9% |
| D | 0.01% - 19.9% |
| F | 0% 或失败 |

**Full Combo (FC)**（全连）是指没有获得 W5 或 Miss 判定。

### 生命条类型

选择您偏好的生命条行为：

- **Normal Bar**：传统生命条，良好判定增加生命，失误减少生命
- **Battery**：固定生命数（默认 3）。每次连击中断消耗一条生命。失去所有生命即失败。
- **Survival**：失误时生命下降更快。更难恢复。

在 **OPTIONS** 顶部选择 **「游戏」** 分类中设置。

### 音符类型

Best-StepMania 支持多种音符类型：

| 类型 | 描述 |
|------|------|
| **Tap**（单键） | 标准音符。当音符到达判定区时按下。 |
| **Hold**（长条） | 按住直到音符尾部通过判定区。提前松开会失去分数。 |
| **Roll**（滚键） | 类似长条，但需要反复敲击而非按住。停止敲击则滚键中断。 |
| **Mine**（地雷） | 避开这些。触发地雷会中断连击并减少生命条。 |
| **Lift**（抬键） | 当音符到达判定区时松开按键（与单键相反）。 |
| **Fake**（假键） | 仅视觉效果的音符，不影响评分。用于视觉装饰。 |

### 游戏修改器

开始歌曲前，您可以在 **Player Options** 界面应用修改器：

| 修改器 | 效果 |
|--------|------|
| **Speed** | 调整滚动速度。"C" 值表示每秒固定像素数。"x" 值乘以 BPM。 |
| **Reverse** | 音符从上到下滚动，而非从下到上。 |
| **Mirror** | 水平翻转音符列。 |
| **Sudden** | 音符在屏幕上半部分隐藏。 |
| **Hidden** | 音符在屏幕下半部分隐藏。 |
| **Rotate** | 音符滚动时旋转。 |
| **Note Scale** | 调整音符的视觉大小。 |

### 音符样式与 P1 / P2（Player Options）

在 **Player Options** 中可为每位玩家选择 **音符样式（Note Style）**（箭头或主题化外观等，与当前 NoteSkin 配合）。在 **Pump 单人**模式下可分别为 **P1** 与 **P2** 选择难度/谱面；**Pump 双人**与 **Routine** 模式下，P2 侧部分选项会与 P1 联动锁定。**Routine** 模式还可为双方指定显示配色，便于区分两层音符。

### 合作模式

Best-StepMania 支持多种游戏模式：

- **Solo**：单人使用单面板（5 轨道）
- **Double**：单人使用双面板（10 轨道）
- **Routine**：两名玩家共享一个谱面，各自击打不同音符
- **Co-op**：两名玩家同时在独立面板上游戏

## 5. 编辑器

内置谱面编辑器允许您创建和编辑步进谱面。

### 打开编辑器

1. 在标题画面中，选择 **CHART EDITOR**
2. 选择要编辑的歌曲，或创建新歌曲
3. 如果歌曲没有谱面，使用 **New Chart** 按钮创建

### 界面布局

编辑器分为 **顶部工具栏**（播放/暂停预览、撤销与重做、保存、量化、当前音符类型、网格/节拍线显示开关、Routine 层切换等）、**左侧边栏**（多页签：歌曲元数据、谱面属性、BPM 与时间轴相关面板、音符统计等，会随谱面类型变化）、**中央谱面网格**（判定线、节拍与小节线）以及 **底部状态栏**（缩放、当前节拍位置、保存结果提示等）。编辑 **Routine** 谱时，可用工具栏或快捷键在层之间切换。

### 添加和编辑音符

**放置音符：**

1. 从工具栏选择所需的音符类型
2. 在网格的所需位置点击
3. 音符会吸附到当前的量化设置

**创建长条音符：**

1. 选择 Hold 或 Roll 类型
2. 从开始位置点击并拖动到结束位置
3. 松开以创建长条

**删除音符：**

- 右键点击音符，或
- 选择音符并按 `Delete`

**选择多个音符：**

- 点击并拖动创建选择框
- 按住 `Shift` 并拖动进行框选
- 按住 `Ctrl` 并右键拖动添加到选择

### BPM 和判定

左侧的 **BPM / Timing** 面板允许您：

- 查看现有的 BPM 变化
- 在特定节拍添加新的 BPM 变化
- 删除 BPM 变化

添加 BPM 变化：

1. 输入变化发生的节拍数
2. 输入新的 BPM 值
3. 点击 **Add BPM change**

### 保存谱面

谱面以 `.sm` 或 `.ssc` 格式保存。保存方法：

1. 点击 **Save** 按钮，或按 `Ctrl+S`
2. 编辑器自动保存到歌曲的谱面文件
3. 保存成功时显示确认消息

您还可以：

- **Duplicate Chart**：创建当前谱面的副本，包含所有音符和判定
- **Delete Chart**：删除当前谱面（需要确认）

### 快捷键参考

完整的快捷键列表请参阅 [编辑器快捷键](#编辑器快捷键) 部分。

## 6. 设置

在标题画面选择 **OPTIONS** 后，通过顶部**分类标签**切换不同设置页。

**常规**：界面语言、主题；此页底部提供 **将所有设置重置为默认值**（需确认）。

**内容管理**：进入 **管理歌曲包**（导入包、编辑包内歌曲等），与从标题画面 **OPTIONS** 进入歌曲包管理相同。

**音频**：主音量、音乐音量、音效音量；**节奏音效**的开关、音量、**风格**（旁有试听）；**界面音效**风格（可试听）；**音频偏移 (ms)**。

**显示**：窗口与显示模式、垂直同步、目标帧率、UI 缩放；**光标**（样式预设、缩放、不透明度、光晕等）；**点击涟漪**开关及持续时间、缩放与透明度等参数。

**游戏**：**双人面板间距**（16–160 像素）；**判定显示风格**（DDR / ITG）、游戏中是否**显示判定偏移**、**生命条类型**与 Battery **命数**；**显示粒子特效**（性能紧张时可关闭）。

**键盘**：十条 Pump 双人轨道的击打键位，以及全局/游玩/编辑器/Player Options 等**快捷键**；均可单独重绑，并可用 **重置所有键位** 恢复默认。

**关于**：版本与简介；**导出诊断** 可将环境信息导出为文件（完成后通常会打开所在文件夹），便于排错或反馈。

**说明**：**自动播放 (Auto Play)** 在选歌之后进入的 **Player Options（玩家选项）** 页面中开关，不在 **OPTIONS** 设置分类里。

## 8. 故障排除

### 常见问题

#### 歌曲列表中没有显示歌曲

**解决方案：**

1. 检查您的歌曲文件格式是否正确（.sm 或 .ssc）
2. 确保每首歌曲都有音频文件（.ogg、.mp3 或 .wav）
3. 确认歌曲位于应用会扫描的目录，或通过 **OPTIONS** > **内容管理** > **管理歌曲包** 导入包
4. 在歌曲包界面尝试刷新或重新进入选歌

#### 音频与音符感觉不同步

**解决方案：**

1. 进入 **OPTIONS** > **音频**
2. 调整 **Audio Offset** 值：
   - 如果音符感觉早，使用正值（延迟音频）
   - 如果音符感觉晚，使用负值（提前音频）
3. 在 **OPTIONS** > **游戏** 中启用 **Show Timing Offset** 查看您的判定时间
4. 从小调整开始（±10-20ms）并微调

#### 游戏运行缓慢或卡顿

**解决方案：**

1. 降低 **Target FPS** 设置（**OPTIONS** > **显示**）
2. 在 **OPTIONS** > **游戏** 中关闭 **显示粒子特效**
3. 尝试更小的 **Note Scale** 以减少渲染负载
4. 确保启用 VSync 以防止帧率不稳定
5. 关闭其他应用程序以释放系统资源

#### 谱面加载失败

**解决方案：**

1. 检查谱面文件是否有效（尝试在其他 StepMania 客户端中打开）
2. 确保谱面中引用的音频文件存在
3. 检查歌曲文件夹的文件权限
4. 在开发者控制台中查找错误消息（游戏中按 `F3`）

#### 键盘输入无响应

**解决方案：**

1. 检查游戏窗口是否有焦点
2. 在 **OPTIONS** > **Keyboard** 中验证您的键位绑定
3. 某些键盘有按键冲突限制；尝试使用不同的按键
4. 禁用可能干扰的键位重映射或宏软件

### 获取诊断信息

如果遇到问题，可以：

1. 游戏期间按 `F3`（默认）打开开发者面板，查看 FPS、帧时间、音频同步、粒子数、活动音符数等。
2. 在 **OPTIONS** > **关于** 中点击 **导出诊断**，生成诊断文件并便于附带在反馈中。

### 重置为默认值

如果设置损坏或您想重新开始：

1. 进入 **OPTIONS**，打开 **「常规」** 分类
2. 点击 **将所有设置重置为默认值** 并确认
3. 游戏将恢复出厂默认值并重新扫描默认歌曲文件夹
