# Best-StepMania v1.3.0

**发布日期：2026-04-15**

## 玩法更新

- **游玩模式切换**：选歌界面新增游玩模式切换条（单人 / 双人等），选择后直接带入游玩会话，不再需要额外操作。
- **节拍线渲染优化**：游玩时的节拍线视觉效果更清晰，支持 BPM 变速段正确显示。
- **音符皮肤与判定框刷新**：音符绘制逻辑更新，皮肤渲染与判定时间窗口显示更一致。
- **玩家选项界面更新**：PlayerOptionsScreen 布局与内容调整，各选项分区更清晰。
- **节奏音效（SFX）设置**：选项界面新增节奏音效开关与快照配置，可单独控制判定音和菜单音效。
- **启动进度与取消**：应用启动覆盖层新增加载进度指示与取消按钮，长时间加载时可中断。
- **设置实时同步**：设置变更后自动同步至后端，无需手动保存。

## Bug 修复

- 修复 Miss 判定时 DP 扣分未正确计算为零的问题，Miss 默认不再产生额外 DP 扣罚。
- 修复 DP 计算与地雷（Mine）判定时序未对齐评分配置的问题，判定结果现与评分系统配置完全一致。
