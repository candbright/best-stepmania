# Best-StepMania v1.2.0

发布日期：2026年4月14日

## 玩法更新

- 选歌预览与播放相关流程加强，进度与状态反馈更稳定。
- 判定与音符场表现、游玩中反馈路径打磨，读谱与击打体感更一致。
- 统一弹窗样式，并改进数字输入与下拉选择，选歌筛选、歌曲包导入等表单更好用。
- 设置中可配置更多按键，并在实际游玩中按新配置生效。
- 编辑器侧栏与工具栏整理，常用编辑动作更集中。
- 筛选或排序变化后，当前选中曲目与列表展示保持同步，减少选中和列表「各说各话」的情况。
- 应用外壳与窗口视口对齐，全屏与缩放时布局更贴边。

## Bug 修复

- 改善指针在输入控件上的状态，以及 Esc 关闭弹窗与返回上一层级的行为。
- 节拍器音效动态余量上调，减轻容易顶满、听感发闷的情况。
