# Best-StepMania v1.3.5

发布日期：2026-04-18

## 玩法更新

- 在 **OPTIONS** → **显示** 中可开启全局 **FPS** 显示，便于观察帧率。
- 尚未选中歌曲时，选歌与标题等场景会用随机起点预热预览队列；切换游玩模式且当前没有选中曲目时，更容易直接进入随机曲目。
- 编辑器预览：带正 **OFFSET** 的谱面在预览时音轨与谱面时间对齐更准确；使用随 BPM 变化的滚动方式时，含停拍或延时段的谱面流速与节拍线显示更一致。
- 主选歌界面左侧歌曲列表的最小宽度改为固定默认值，避免被拖到异常过窄的宽度。

## Bug 修复

- 修复从编辑器预览进入游玩后，结算页 **重试** 无法从同一预览位置再次开始的问题。
- 修复从编辑器返回选曲界面，或从游玩返回选歌页后，预览音乐有时不恢复的问题。
